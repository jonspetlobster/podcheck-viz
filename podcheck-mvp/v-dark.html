How PodCheck verifies claims made on podcasts.

1. Claim Extraction

We use AI to transcribe podcast episodes and identify verifiable factual claims ‚Äî statements about data, events, science, or history that can be checked against evidence. Opinions, predictions, and subjective statements are excluded.

2. Evidence Gathering

Each claim is cross-referenced against multiple authoritative sources:

  • Peer-reviewed scientific papers and meta-analyses
  • Government data (BLS, CBO, Fed, CDC, WHO)
  • Established news organizations with fact-checking teams
  • Academic institutions and expert consensus
  • Primary sources cited by the speaker

3. Verdict Scale

  • True ‚Äî Fully supported by available evidence
  • Mostly True ‚Äî Largely accurate with minor caveats
  • Mixed ‚Äî Contains both accurate and inaccurate elements
  • Mostly False ‚Äî Significant inaccuracies
  • False ‚Äî Contradicted by available evidence

4. Accuracy Grades

Each speaker receives a letter grade (A‚ÄìF) based on their overall accuracy rate. True = 100%, Mostly True = 80%, Mixed = 50%, Mostly False = 20%, False = 0%. Grades: A ‚â• 85%, B ‚â• 70%, C ‚â• 55%, D ‚â• 40%, F < 40%.

5. Transparency

Every verdict includes the original quote, context, sources consulted, and reasoning. We show our work so you can evaluate our conclusions independently.
